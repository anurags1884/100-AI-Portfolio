# 🧮 Professional Smart Calculator

A modular Python Smart Calculator developed as part of my **100 AI Portfolio Projects**.

---

## 📌 Project Overview

This calculator is designed using professional software engineering practices. Instead of placing all code in a single file, the application follows a modular architecture with separate files for operations, validation, history management, and constants.

---

## 🚀 Features

### Basic Operations

- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Power
- Floor Division

### Scientific Operations

- Square Root
- Factorial
- Logarithm
- Trigonometric Functions

### Advanced Features

- Input Validation
- Exception Handling
- Calculation History
- Save History to JSON
- Read Previous History
- Colored Terminal Output

## ▶️ How to Run

Clone the repository:

```bash
git clone https://github.com/AnuragShinde04/100-AI-Portfolio.git
```

Go to the project folder:

```bash
cd python/01_Calculator
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
python calculator.py
```
